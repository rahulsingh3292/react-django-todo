Msg -> endpoint -> ApiMethod
  • login -> /api/login/ -> post

  • register -> /api/register/ -> post

  • get all-todos -> /api/get-todos/ -> get

  • add todo -> /api/add-todo/ -> post

  • edit or update-todo -> /api/edit-todo/todo_id:int/ -> patch

  • delete todo -> /api/delete-todo/todo_id:int/ -> delete
